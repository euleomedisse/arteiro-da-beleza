# arteiro-da-beleza

🌿 Arteiro da Beleza | Studio Afro & Escola
"Nossa Arte é a sua Coroa."

Bem vindo ao repositório oficial do Arteiro da Beleza, um Studio Afro localizado no coração de Salvador, Bahia. Este projeto é uma plataforma imersiva que celebra a estética ancestral, promove o empreendedorismo preto e oferece serviços especializados de tranças e dreadlocks.

🎨 Identidade e Estética (Afro Chic)
O design foi projetado para transmitir sofisticação e pertencimento, utilizando elementos que remetem à terra e à natureza:

Cores: Predominância do Verde Marca (#1e432d) aliado a tons terrosos como o Terracota (#c35b3e) e o Palha (#f8f3ed).

Tipografia: Uso das fontes Fraunces (com curvas rítmicas e artesanais) e Bricolage Grotesque (modernidade e leitura fluida).

Imersão: Efeitos de Parallax para profundidade visual e animações de entrada (AOS.js) que ditam o ritmo da navegação.

🛠️ O que compõe o Projeto
O site foi estruturado para ser uma ferramenta de conversão direta e educação:

Bio Imersiva: Apresentação do fundador Luan e a filosofia do studio.

Serviços de Realeza: Divisão entre cuidados estéticos (Tranças/Dreads) e Formação Profissional.

Agô Ori Cosméticos: Vitrine exclusiva para o shampoo específico de dreadlocks com venda direta via WhatsApp.

Portfolio Dinâmico: Carrossel de resultados com autoplay que exibe a "coroa" de clientes reais.

Hub de Parcerias: Espaço dedicado a impulsionar novos negócios e ideias pretas.

🚀 Tecnologias e Performance
Foco total na experiência Mobile First, já que a maior parte do público acessa via celular:

HTML5 & CSS3: Uso de variáveis customizadas e Grid Layout.

JavaScript Vanilla: Lógica de scroll, carrossel automático e navegação.

AOS Library: Biblioteca para animações de rolagem.

GitHub Pages: Hospedagem estável e segura.
